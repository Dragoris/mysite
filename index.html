About Me

Hey, my name is Matt and I've become fascinated by the internet and it's ability to facilitate shared ideas across the world. For years I enjoyed everything the web had to offer without ever questioning why or how it all worked, but once I started drilling down into those questions, that's when I got hooked. After studying up on how all the pieces fit together I realized it wasn't done through some unattainable sorcery (even though it still sometimes feels like it!), there are a set of tools anyone can learn to make something amazing, if they have the persistence to put in the time and effort required. This site is where I'll be sharing what I've make and what I've learned, I hope you enjoy.

When I'm not digging into the internet you'll find me playing basketball with my team, marveling at the Warriors' unselfish style of play and chemistry, or backpacking through the northern California wilderness with friends and family.

LazyDJ

LazyDJ is a music playing app that I built using SoundCloud's API to search for and play songs. I decided to use jQuery UI for a search bar so that the processed results can be displayed automatically and updated asynchronously for a seamless user experience.
